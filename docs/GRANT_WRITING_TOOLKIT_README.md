## Critical Action Items Before Applying

### Priority 1: MUST DO (Red Flag if Missing)

- [ ] **Get programmed in state STIPs** (Iowa, Ohio, Pennsylvania, Nebraska)
- [ ] **Complete benefit-cost analysis** (target B/C > 5.0)
- [ ] **Secure 100% committed O&M funding** for 5 years
- [ ] **Execute MOUs with all partner states** (specific deliverables, dollar commitments)

### Priority 2: HIGHLY IMPORTANT (Significantly Strengthen Application)

- [ ] **Get letters of intent from end users** (Google, Waze, Apple Maps)
- [ ] **Document existing system benefits** (Iowa I-80 before/after analysis)
- [ ] **Quantify baseline data** (crashes, delays, economic costs for all corridors)
- [ ] **Get letters from freight industry** (ATA, OOIDA, major carriers)

### Priority 3: STRENGTHEN (Good to Have)

- [ ] Get included in state freight plans
- [ ] Get letters from fleet management providers (Trimble, Omnitracs)
- [ ] Get letters from congressional delegation
- [ ] Complete academic partnership for evaluation

**Timeline:** Start 6-12 months before target grant deadline

## Budget Guidance

### Recommended Project Size for Corridor Communicator

**Small Project:** $5M - $10M
- 3-5 states, 1,000-1,500 miles
- 50-100 connected work zones
- Match: $2.5M-$5M

**Medium Project:** $10M - $25M (RECOMMENDED)
- 4-6 states, 2,000-3,000 miles
- 150-250 connected work zones
- Advanced V2X, navigation integration
- Match: $5M-$12.5M

**Large Project:** $25M - $50M
- 6-10 states, 4,000+ miles
- 300+ connected work zones
- National data platform
- Match: $12.5M-$25M

**Recommendation:** Target **Medium Project ($15M-$20M total)** for highest competitiveness

### Budget Allocation (Rule of Thumb)

- Program Management: 10%
- Digital Infrastructure: 35%
- Field Equipment: 40%
- Integration & Applications: 10%
- Contingency: 5%

## Common Questions

### Q: How long does it take to develop a competitive application?
**A:** 6-12 months for a strong application. Can be accelerated to 3-4 months if partnerships and programmatic support already in place.

### Q: What's the most important factor for success?
**A:** Being programmed in state/regional plans. SAAFIR failed partly because it wasn't in any state STIP, freight plan, or long-range plan.

### Q: What B/C ratio should we target?
**A:** Minimum 3.0 to be competitive. Target 5.0+ for highly competitive. SAAFIR had 13.3 but still wasn't selected (other issues).

### Q: How many partners is ideal?
**A:** 3-5 committed partners > 10+ aspirational partners. SAAFIR had 11 states which was too complex.

### Q: Should we apply if not fully ready?
**A:** No. Better to wait one cycle and submit a strong application than rush a weak one. Resubmissions have lower success rates.

### Q: What if we don't have pilot data?
**A:** Use peer-reviewed research and CMFs from FHWA Clearinghouse, but pilot data significantly strengthens application. Corridor Communicator has Iowa I-80 data - use it!
